Usage: pdq-planner-<version>.jar [options]
  Options:
    -c, --config
       Directory where to look for configuration files. Default is the current
       directory.
    -h, --help
       Displays this help message.
       Default: false
    -o, --output
       Output file name.
       Default: out.xml
  * -q, --query
       Path to the input query definition file.
  * -s, --schema
       Path to the input schema definition file.
    -v, --verbose
       Activates verbose mode.
       Default: false
    -D
       Dynamic parameters. Override values defined in the configuration files.
       Syntax: -Dkey=value
       Default: {}
